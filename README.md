# Amazon Price Tracker
### This is a simple python project which can be used to track a products prices on [**Amazon**]([https://www.amazon.com](https://www.amazon.com/)). It uses Bs4 and Requests library to do the task. 

## Modules Used
- Requests 
- BS4 
- Pygame 
- Time
- JSON
